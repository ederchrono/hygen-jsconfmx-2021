# hygen-jsconfmx-2021

Simple Node/Express repository to run Hygen prompts, templates and code injectors.

## Setup
Just run:

```npm install```

## Usage
Run the local server with 

```npm start```

Visit `http://localhost:3000/` to see the site

### Create a new page

Run:

```npm run new:page```

and follow the prompt

### Create a new API endpoint

Run: 

```npm run new:api```

and follow the prompt
